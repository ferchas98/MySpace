hipervínculo

La <a> etiqueta define un hipervínculo, que se utiliza para vincular de una página a otra.
El atributo más importante del <a>elemento es el href atributo, que indica el destino del enlace.
De forma predeterminada, los enlaces aparecerán de la siguiente manera en todos los navegadores:
Un enlace no visitado está subrayado y en azul. Un enlace visitado está subrayado y en color violeta. Un enlace activo está subrayado y en rojo.

Ejemplo de uso

<a> href="https://www.w3schools.com">Visit W3Schools.com!</a>

artículo

Un artículo debe tener sentido por sí solo y debe ser posible distribuirlo independientemente del resto del sitio.

Ejemplo de uso

<article> <h2>Google Chrome</h2> <p>oogle Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!</p> </article>

aparte

El contenido aparte debe estar indirectamente relacionado con el contenido circundante.

Ejemplo de uso

<aside> <h4>Epcot Center</h4> <p>Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.</p> </aside>

audio

La <audio> etiqueta se utiliza para incrustar contenido sonoro en un documento, como música u otras transmisiones de audio.

Ejemplo de uso

<audio controls> <source src="horse.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio>

encabezados

Las etiquetas <h1> to <h6> se utilizan para definir encabezados HTML.
<h1> Define el título más importante.
<h6> Define el encabezado menos importante.
Nota: Utilice solo uno <h1> por página; este debe representar el título/asunto principal de toda la página. Además, no se salte los niveles de encabezado: comience con <h1>, luego use <h2>, y así sucesivamente.

Ejemplo de uso

<h1> encabezado </h1>
